## Required tags for AWS resources

When creating AWS resources you must attach the following tags where possible.

| Tag                     | Value                |
|-------------------------|----------------------|
| `gl_realm`              | `sandbox`            |
| `gl_env_name`           | `<aws_account_name>` |
| `gl_env_type`           | `experiment`         |
| `gl_owner_email_handle` | `ai_agents`          |
